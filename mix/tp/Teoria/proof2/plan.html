Ejemplo 2: Plan de prueba

Esquema de la prueba 


 { PRE: x0 = x and y0 = y }

    if (x > y) then y := x

               else x := y

 { POST: x = y = max(x0,y0) }
  1. Probar |= { PRE and x > y } y := x { POST }
  2. Probar |= { PRE and ~(x > y) } x := y { POST }
  3. concluir que |= { PRE } if-then-else { POST }
