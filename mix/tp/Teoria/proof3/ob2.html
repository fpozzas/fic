Demostrar que 

 INV and (k < b) ==> P3
O sea, que si suponemos 
(*)    a<=k<=b and forall i, a <= i <  k : x[i]=x0[i+1]
                   and forall i, k <= i <= b : x[i]=x0[i]
                   and (k < b)
tenemos que probar 
a <= k+1 <= b  
    and forall i, a <= i <  k+1 : x[x[k+1]/k+1-1][i] = x0[i+1]   
        and forall i, k+1 <= i <= b : x[x[k+1]/k+1-1][i] = x0[i]
En efecto: 
  •   (k < b) and a<=k<=b ==> a<=k < b
      a<=k ==> a <= k+1 y k < b ==> k+1 <= b.
      por lo tanto ya hemos probado que a <= k+1 <= b que
      es la primera cláusula de P3.

  •   Ahora, forall i, a <= i < k :x[x[k+1]/k][i]=x[i] (pues i es distinto
      de k), pero por la hipótesis (*) este x[i] es igual a x0[i+1]
      al ser i < k. Si i=k entonces x[x[k+1]/k][i]=x[i+1]
      =x0[i+1] por la tercera cláusula de (*). Esto prueba la segunda
      cláusula de P3.

  •   Finalmente, si k+1 <= i <= b entonces k <= i <= b y además i es
      diferente de k y x[x[k+1]/k][i]=x[i] que es igual a x0[i] por la
      tercera cláusula de (*).
    QED
