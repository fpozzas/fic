P2: INV con x reemplazado por x[x[k]/k-1]

El array x[x[k]/k-1] representa el array que es igual que x salvo en k-1 donde contiene x[k] (o sea lo que contiene x en el lugar k). 
    a <= k <= b  
    and forall i, a <= i <  k : x[x[k]/k-1][i] = x0[i+1]
    and forall i, k <= i <= b : x[x[k]/k-1][i] = x0[i]

P3: P2 con k reemplazada por k+1

    a <= k+1 <= b  
    and forall i, a <= i <  k+1 : x[x[k+1]/k+1-1][i] = x0[i+1]   
    and forall i, k+1 <= i <= b : x[x[k+1]/k+1-1][i] = x0[i]
