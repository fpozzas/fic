Consideremos este programa que hace un desplazamiento a la izquierda de los elementos de un array.
Los índices van de a hasta b, x[a] es alterado, pero x[b] queda igual 


 { PRE: a <= b and x = x0 }

   k := a;

   while k < b do

     (k := k + 1;

     x[k-1] := x[k])


 { POST: forall i, a <= i < b : x[i] = x0[i+1] }
Probar que este programa es correcto respecto a las especificaciones dadas PRE y POST.
