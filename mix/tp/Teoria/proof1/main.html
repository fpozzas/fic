Ejemplo de prueba 1: Programa que calcula el resto

  • Hacer prueba
  • Plan de prueba

    • Consideremos el siguiente programa que se supone que calcula el resto. (Sólo en este ejemplo usaremos mayúsculas para las variables mutables --que son de Loc--y minúsculas para las variables numéricas --que son de Intvar.) 

    
   { PRE: x >= 0 }
     
       Q := 0;
       
       R := x;

       while R >= y do

         (R := R - y;

         Q := Q + 1)


   { POST: 0 <= R < y  and  x = Q * y + R }
    Probar que esta aserción de corrección parcial es correcta. O lo que es lo mismo, que el programa es correcto respecto a las especificaciones PRE y POST.
